## 产品概述

### 产品定位
AI法律助手是一个现代化、智能化的法律案件管理系统，为律师和法务人员提供全方位的案件管理和AI辅助功能。

### 目标用户
- 律师事务所律师
- 企业法务人员
- 法律服务机构

### 核心价值
- **效率提升**: 自动化案件信息管理，减少重复劳动
- **智能辅助**: AI驱动的分析和建议，提升决策质量
- **数据可视化**: 直观展示案件关系和时间线
- **知识管理**: 法律研究和案例检索一体化

## 功能详细描述

### 1. 用户认证与管理

#### 1.1 用户登录
**功能描述**: 用户通过邮箱和密码登录系统

**交互流程**:
1. 用户输入邮箱和密码
2. 点击"登录"按钮提交
3. 系统验证凭据
4. 登录成功后跳转到案件列表

**验证规则**:
- 邮箱格式验证
- 密码长度验证（最小6位）

#### 1.2 用户注册
**功能描述**: 新用户通过邮箱注册账号

**交互流程**:
1. 用户输入邮箱、密码、确认密码
2. 点击"注册"按钮
3. 系统创建账号并发送验证邮件
4. 用户验证邮箱后自动登录

**验证规则**:
- 邮箱格式验证
- 密码强度验证
- 两次密码一致性验证

#### 1.3 忘记密码
**功能描述**: 用户可以通过邮箱重置密码

**交互流程**:
1. 点击登录页"忘记密码？"链接
2. 输入注册邮箱
3. 点击"发送重置链接"
4. 系统发送密码重置邮件
5. 用户通过邮件链接重置密码

#### 1.4 个人资料管理
**功能描述**: 用户可以查看和编辑个人信息

**可编辑字段**:
- **基本信息**: 姓名、职位、邮箱、手机号码、所属部门、办公地点、个人简介
- **账号安全**: 当前密码、新密码、确认新密码
- **偏好设置**: 邮件通知、短信通知、语言、主题

### 2. 案件管理

#### 2.1 案件列表
**功能描述**: 以表格形式展示所有案件

**显示字段**:
- 案件名称（主标题）
- 案件编号
- 客户名称
- 案件类型
- 状态标签（进行中、草稿、已结案）
- 负责人
- 最后更新时间

**操作功能**:
- 查看详情
- 编辑案件
- 结案（仅进行中案件）
- 删除案件

#### 2.2 新建/编辑案件表单

**表单分区**:

##### 基础信息
| 字段名称 | 字段类型 | 是否必填 |
|---------|---------|---------|
| 案件名称 | 文本 | 是 |
| 案件编号 | 文本 | 否 |
| 案由 | 下拉选择 | 否 |
| 案件阶段 | 下拉选择 | 否 |
| 管辖法院/仲裁委 | 文本 | 否 |
| 承办法官 | 文本 | 否 |
| 立案日期 | 日期选择器 | 否 |

##### 当事人信息
- **原告**: 姓名、主体类型、身份证号、联系电话、地址
- **被告**: 姓名、主体类型、身份证号、法定代表人、对方律师、地址
- **第三人**: 姓名、主体类型、身份证号、地址

##### 案情描述
| 字段名称 | 字段类型 |
|---------|---------|
| 案情摘要 | 多行文本 |
| 争议焦点 | 文本 |
| 客户诉求 | 多行文本 |

##### 财务信息
| 字段名称 | 字段类型 |
|---------|---------|
| 诉讼标的额 | 数字 |
| 律师费报价 | 数字 |
| 预估诉讼费 | 数字 |
| 计费时长 | 数字 |

##### 联络人信息
| 字段名称 | 字段类型 |
|---------|---------|
| 姓名 | 文本 |
| 职位/角色 | 文本 |
| 电话 | 文本 |
| 邮箱 | 邮箱 |
| 地址 | 文本 |

#### 2.3 案件详情页
**布局结构**:
- 顶部: 面包屑导航
- 左侧: 标签页导航
- 右侧: 内容区域

**标签页**:
1. 基础信息
2. 案情描述
3. 当事人信息
4. 证据管理（证据清单 + 证据分析规划）
5. 财务信息
6. 高级功能（AI分析、AI助手、关系图谱、时间轴、报价生成）

### 3. 证据管理

#### 3.1 证据上传
**功能描述**: 支持拖拽或点击上传多个文件

**支持格式**: PDF, Word, Excel, JPG, PNG, GIF

**交互方式**:
- 点击上传区域选择文件
- 拖拽文件到上传区域
- 显示上传进度
- AI自动识别并分类

#### 3.2 AI证据分析

**证据完整度评估**:
- 完整度百分比
- 已收集/缺失证据数量
- 证据优先级标签（极高、高、中、低）

**导出功能**: 导出证据清单为Excel或PDF

---

### 4. AI智能功能

#### 4.1 AI智能分析
**功能描述**: AI基于案情和证据进行综合分析

**分析内容**:
- 胜诉率百分比预测
- 风险点识别（高风险/一般提示）
- 诉讼策略建议

#### 4.2 AI对话助手
**功能描述**: 基于案件材料的智能问答

**快捷建议**:
- "分析本案的胜诉概率"
- "生成一份证据收集清单"
- "起草一份律师函"
- "查找类似的判决案例"

#### 4.3 关系洞察
**功能描述**: D3.js展示人物和公司关系网络

**显示内容**:
- 节点: 人物（圆形）、公司（方形）
- 连线: 关系类型 + 金额
- 交互: 拖拽、缩放、点击查看详情

#### 4.4 证据时间轴
**功能描述**: AI生成的证据事件时间线

**显示内容**:
- 时间线（垂直布局）
- 事件节点: 日期、标题、描述、关联证据

#### 4.5 报价生成器
**功能描述**: 自动生成法律服务报价书

**自动填充**: 案件名称、客户名称、案件类型、诉讼标的额
**导出**: PDF格式报价书

### 5. 法律检索

#### 5.1 检索功能
**搜索类型**: 案例检索、法规检索

**案例筛选条件**:
- 法院层级（最高法、高级、中级、基层）
- 地区
- 裁判年份
- 审理程序（一审、二审、再审、执行）
- 文书类型（判决书、裁定书、调解书）

**法规筛选条件**:
- 法规类别（法律、行政法规、司法解释）
- 发布机关
- 效力级别
- 时效性（有效、失效）

#### 5.2 详情查看

**案例详情**: 案件基本信息、当事人信息、审理经过、事实认定、裁判理由、裁判结果

**法规详情**: 法规基本信息、法规正文、章节导航

### 6. 合同审查

**核心功能**:
- **智能风险分析**: AI自动识别风险点并提供修改建议。
- **在线预览与编辑**: 支持 Word 文档在线预览，并集成在线编辑器（支持阿里云 WebOffice/WPS 方案）实现原文修改。
- **修订交互**: 用户可对 AI 建议进行“采纳”、“拒绝”或“手动修改”操作，系统自动将采纳的建议应用到文档中。
- **多样式导出**: 支持导出多种格式的审查成果。

**交互流程**:
1. **上传**: 上传合同文件（支持 Word/PDF）。
2. **分析**: 选择合同类型与审查立场（甲方/乙方/中立），启动 AI 审查。
3. **审查与编辑**:
    - **分栏视图**: 左侧预览合同原文，右侧显示风险卡片。
    - **定位高亮**: 点击风险卡片，左侧文档自动跳转并高亮对应条款。
    - **修订操作**: 针对每个风险点，用户可点击“一键采纳”AI 建议，或在右侧进行二次编辑后采纳。
    - **全文编辑**: 并在需要深度修改时，支持调用在线编辑器进行全文自由修订。
4. **导出**:
    - **修订版**: 包含所有修订痕迹（Track Changes）的 Word 文档。
    - **法律意见书**: 自动生成的正式审查报告（含风险汇总与专业建议），支持 PDF 下载。
    - **纯净版**: 接受所有修订后的最终 Word 文档。

**审查结果展示**:
- **风险概览**: 综合评分、高/中/低风险数量统计。
- **逐条详情**: 原文条款、风险说明、依据法规、AI 修改建议（支持红字修订显示）。
