## 一、项目背景

### 我们在做什么？

我们正在开发一个**AI法律助手系统**，目标是帮助律师团队更高效地管理案件、整理证据、分析案情。

目前系统的所有功能都是**开发团队根据自己的理解**设计的，我们需要您的反馈来确认：
- ✅ 哪些功能是您真正需要的
- ❌ 哪些功能不太实用
- 💡 还缺少哪些重要功能

## 二、现有功能说明

### 📁 模块一：案件管理

#### 1.1 案件列表
**这是什么？**  
就像一个案件档案柜，可以看到所有案件的基本信息。

**能做什么？**
- 看到所有案件的名称、编号、客户、状态
- 搜索案件（输入案件名或客户名就能找到）
- 点击案件进入详情页

**实际场景举例**：  
比如您想找"张三诉李四借款纠纷"这个案子，直接在搜索框输入"张三"或"借款"就能找到。

**💬 请反馈**：
- [ ] 这个功能实用吗？
- [ ] 还需要增加什么筛选条件？（比如按时间、按案件类型）
- [ ] 列表里显示的信息够用吗？

---

#### 1.2 新建案件
**这是什么？**  
创建一个新案件档案，把案件信息录入系统。

**需要填写什么？**
1. **基础信息**：案件名称、案由、法院、立案日期等
2. **当事人信息**：原告、被告的详细信息（支持多个）
3. **案情描述**：案件的基本情况、争议焦点、客户诉求
4. **财务信息**：标的额、律师费、诉讼费
5. **联络人信息**：主要联系人的电话、邮箱、地址

**保存方式**：
- 点击"保存" → 案件为"草稿"状态
- 点击"提交" → 案件为"进行中"状态

**实际场景举例**：  
接到一个新案子，您可以把客户提供的所有信息一次性录入，以后随时查看。

**💬 请反馈**：
- [ ] 这些字段够用吗？还缺什么？
- [ ] 有没有不需要的字段？
- [ ] 填写流程会不会太复杂？

---

#### 1.3 案件详情页
**这是什么？**  
点开一个案件后，可以看到这个案件的所有详细信息，分成不同的标签页。

**包含哪些内容？**
- **基础信息**：案件的基本情况
- **案情描述**：详细的案情和诉求
- **当事人信息**：原告、被告、第三人的详细资料
- **证据管理**：上传的证据文件和AI分析
- **财务信息**：费用相关信息
- **高级功能**：AI分析、AI助手、关系图谱、时间轴

**实际场景举例**：  
开庭前想快速了解案情，点开案件详情，所有信息一目了然。

**💬 请反馈**：
- [ ] 信息分类合理吗？
- [ ] 有没有常用信息找不到的情况？
- [ ] 需要调整标签页的顺序吗？

---

#### 1.4 快速编辑
**这是什么？**  
不用打开完整的编辑页面，直接在详情页点"笔"图标就能修改某一部分信息。

**能编辑什么？**
- 基础信息（案件名称、法院等）
- 案情描述（案情摘要、争议焦点等）
- 财务信息（标的额、律师费等）
- 联络人信息（联系人电话、邮箱等）

**实际场景举例**：  
客户突然改了诉求，您只需要点开"案情描述"的编辑按钮，改完保存，不用重新填整个表单。

**💬 请反馈**：
- [ ] 这种分区编辑方便吗？
- [ ] 还是更喜欢一次性编辑所有信息？

### 📂 模块二：证据管理

#### 2.1 证据上传
**这是什么？**  
把证据文件（合同、聊天记录、转账凭证等）上传到系统。

**能做什么？**
- 拖拽文件上传（支持PDF、Word、图片等）
- AI自动识别证据类型（合同、支付凭证、往来函件等）
- 显示上传进度

**实际场景举例**：  
客户发来一堆微信聊天记录截图，您直接拖进系统，AI会自动分类为"往来函件"。

**💬 请反馈**：
- [ ] AI分类准确吗？
- [ ] 支持的文件格式够用吗？
- [ ] 需要批量上传功能吗？

---

#### 2.2 AI证据分析
**这是什么？**  
AI根据案件类型，告诉您应该收集哪些证据，哪些证据最重要。

**能看到什么？**
- **证据完整度**：比如显示"已收集3/6，完整度50%"
- **证据清单**：列出所有应该收集的证据
- **优先级**：标注哪些证据是"极高"、"高"、"中"、"低"优先级
- **收集状态**：打勾表示已收集，打叉表示还没收集

**实际场景举例**：  
接到一个买卖合同纠纷，AI会提示您需要收集：买卖合同（极高）、付款凭证（极高）、发货单（高）、聊天记录（中）等。

**💬 请反馈**：
- [ ] AI建议的证据清单实用吗？
- [ ] 优先级判断准确吗？
- [ ] 需要手动添加证据项吗？

### 🤖 模块三：AI智能功能

#### 3.1 AI智能分析
**这是什么？**  
AI看完案情和证据后，给出胜诉率预测、风险提示和策略建议。

**能看到什么？**
- **胜诉率预测**：比如"65%"（大字显示，很醒目）
- **风险点提示**：比如"证据不足"、"诉讼时效临近"等
- **策略建议**：比如"建议补充XX证据"、"可以考虑XX策略"

**实际场景举例**：  
客户问"这个案子能赢吗？"，您可以参考AI的分析给客户一个预期。

**💬 请反馈**：
- [ ] AI分析的准确性如何？
- [ ] 风险提示有用吗？
- [ ] 策略建议实用吗？
- [ ] 需要更详细的分析吗？

---

#### 3.2 AI对话助手
**这是什么？**  
像ChatGPT一样，可以问AI关于这个案件的任何问题。

**能做什么？**
- 问AI问题，比如"这个案子的争议焦点是什么？"
- 使用快捷建议，比如点击"分析胜诉概率"
- 查看历史对话记录

**实际场景举例**：  
准备开庭，想快速了解对方可能的抗辩理由，直接问AI："对方可能会怎么辩护？"

**💬 请反馈**：
- [ ] AI回答的质量如何？
- [ ] 快捷建议实用吗？需要增加哪些？
- [ ] 对话历史需要保存多久？

---

#### 3.3 关系洞察（关系图谱）
**这是什么？**  
用图表的方式展示案件中人物和公司之间的关系。

**能看到什么？**
- 人物和公司的关系网络（谁是谁的老板、谁给谁转账等）
- 资金往来的金额
- 可以拖拽节点、放大缩小

**实际场景举例**：  
复杂的股权纠纷案件，涉及多个公司和个人，用图谱一看就明白谁和谁有关系。

**💬 请反馈**：
- [ ] 这个功能实用吗？
- [ ] 关系图谱需要手动画还是AI自动生成？
- [ ] 需要导出图片功能吗？

---

#### 3.4 证据时间轴
**这是什么？**  
AI根据案件证据生成的时间线，展示证据相关事件的时间顺序。

**能看到什么？**
- 合同签订、付款、交付等关键时间节点
- 每个事件的详细描述

**操作按钮**：
- 刷新：重新让AI生成时间轴
- 编辑：修改时间轴内容
- 导出：导出为PDF或Word

**实际场景举例**：  
想看这个案子的证据链条，打开时间轴，从合同签订到纠纷发生的过程一目了然。

**💬 请反馈**：
- [ ] AI生成的时间轴准确吗？
- [ ] 需要手动添加事件吗？
- [ ] 时间轴需要导出功能吗？
