# 案件管理

案件管理是 Legal Workspace 的核心功能,帮助您系统化地管理所有法律案件。

## 创建新案件

1. 点击左侧导航栏的 **案件管理**
2. 点击页面右上角的 **+ 新建案件** 按钮
3. 填写案件基本信息:
   - 案件名称
   - 案件类型(民事、刑事、行政等)
   - 当事人信息
   - 案件编号
4. 点击 **保存** 完成创建

## 编辑案件信息

1. 在案件列表中找到目标案件
2. 点击案件卡片进入详情页
3. 点击 **编辑** 按钮
4. 修改相关信息后点击 **保存**

## 上传证据材料

1. 进入案件详情页
2. 切换到 **证据** 标签页
3. 点击 **上传证据** 按钮
4. 选择文件并添加证据说明
5. 点击 **确认上传**

支持的文件格式:

- 文档: PDF, Word, Excel
- 图片: JPG, PNG, GIF
- 视频: MP4, AVI

## 案件时间轴

时间轴功能帮助您记录案件的重要节点:

- 自动记录案件创建、修改等操作
- 手动添加重要事项和备注
- 按时间顺序展示案件进展

## 删除案件

⚠️ **注意**: 删除操作不可恢复,请谨慎操作。

1. 进入案件详情页
2. 点击右上角的 **更多** 菜单
3. 选择 **删除案件**
4. 确认删除操作
